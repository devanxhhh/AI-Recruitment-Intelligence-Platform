# AI Recruitment Intelligence Platform

An AI powered recruitment platform that automates resume screening and assists recruiters throughout the hiring process using Google Gemini AI.

## Features

- Resume Parsing
- AI Candidate Evaluation
- Candidate Ranking
- GitHub Profile Analysis
- Assessment Score Integration
- Interview Scheduling
- AI Recruiter Copilot
- CSV Export

## Tech Stack

- Python
- Streamlit
- Google Gemini AI
- Pandas
- GitHub API

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

Create a `.env` file using the variables shown in `.env.example`.

## Project Overview

This application helps recruiters:
- Parse candidate resumes
- Evaluate candidates using AI
- Analyze GitHub profiles
- Combine assessment scores
- Schedule interviews
- Generate AI based hiring recommendations
